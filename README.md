description
===========

Системата се състои от 3 раздела: 
  Търсачка
  Оферти
  - Наеми
  - Продажби
  Брокери
  Клиенти
  
1. Търсачката съдържа следните параметри:
   - Вид на офертата: Наем, Продажба
   - Населено място
   - Цена: от, до, валута
  
2. Офертите съдържат:
   - Вид: Наем/Продажба
   - Заглавие
   - Адрес
   - Наем, валута
   - Квадратура
   - Етаж
   - Описание
   - Квадратура
   - Снимки
   
   Особености:
   - Обзаведен/Необзаведен
   - С паркмясто
   - Асансьор
   - Лукс
   - СОТ
   - Климатик
   - Окабеляване
   - Интернет
   - Видим за външни потребители
   
Опции - добави/изтрий/редактирай

3. Брокери:
   - Име
   - Телефон
   - Потребителско име
   - Парола

Опции - добави/изтрий/редактирай
   
4. Клиенти
   - Име
   - Телефон

Опции - добави/изтрий/редактирай
